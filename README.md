## Installation

To Use Is, simply:

```bash
$ composer require khs1994/is
```

For latest commit version:

```bash
$ composer require khs1994/is @dev
```

## Usage

```php
<?php

require __DIR__.'/vendor/autoload.php';

use Is\Is;

var_dump(Is::is_url('https://www.khs1994.com'));
```
